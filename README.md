VELOURA

Elegant Styles. Timeless Fashion.
VELOURA is a boutique fashion website that showcases curated collections of dresses, bags, and jewellery. The website blends modern silhouettes with artisanal details, creating pieces that feel luxurious and last season after season. This project demonstrates full-stack web development skills including front-end design, responsive layout, and back-end integration.
